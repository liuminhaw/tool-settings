# Setting for OS (Pop!OS)

## Key bindings
### Shortcut for terminals
- kitty
  ```
  super key + k
  ```
- Terminator
  ```
  super key + t
  ```

### Windows shortcut
- Maximize window
  ```
  ctrl + super key + F12
  ```

### Navigation
- Move window one workspace down
  ```
  ctrl + alt + super + Down
  ```
- Move window one workspace up
  ```
  ctrl + alt + super + Up
  ```
